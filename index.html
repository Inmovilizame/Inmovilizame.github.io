Polymer

Webcomponents con esteroides

Google, perdon, Alphabet lleva mucho tiempo poniendo cosas muy interesantes encima de la mesa para el desarrollo web. Una de las que más está dando que hablar son los WebComponents, que de forma nativa proponen la definición de componentes estándar reutilizables de pequeños trozos de HTML (con estilos y funcionalidad incluidos) de forma que puedan ser aprovechados tanto por humanos como por máquinas de una manera eficiente. 

En este marco nace el proyecto Polymer, que acerca este proyecto a casi todos los navegadores y que los WebComponents sean una solución real para el desarrollo web.
